This webpage was made by Aniol Fernàndez Cano, all its code (html, css & javascript) is handmade without the use of any third party templates or libraries.

Remember to log-in in order to unlock all capabilities of the webpage.
You can create a new account or use the user admin with password admin.

I've tried to cover my lack of graphic designer skills with fancy animations and a good polished finish.

The product list can be dynamically modified if a user logs in and upload new software. All changes are reflected into products using local storage.

All pop-ups and animations are handmade using css & javascript.

The browser works by searching by a given set of keywords, displaying a list of products where a match as been made.

The list of products of a search has the matching keywords marked to give to the user the feeling of a good result.

The list of categories is dynamically generated. If you upload a new software with a new category, it will automatically appear on categories window.

When you upload a new product you can add its logo or slide images with an URL of an actual image posted in internet.
